# Abhay Katre Portfolio Website

## Overview
A personal portfolio website built with Mobirise Website Builder. This is a static HTML/CSS/JS website showcasing the skills and projects of Abhay Katre - a Frontend Web Developer, Java Programmer, and B.Tech Student.

## Project Structure
- `index.html` - Main portfolio page
- `assets/` - Static assets directory
  - `bootstrap/` - Bootstrap CSS framework
  - `images/` - Portfolio images and graphics
  - `theme/` - Theme CSS and JavaScript
  - `embla/` - Embla carousel scripts
  - `dropdown/` - Navigation dropdown styles
  - `smoothscroll/` - Smooth scrolling functionality
  - `sociallikes/` - Social media sharing
  - `animatecss/` - CSS animations

## Development
This is a static website served using Python's built-in HTTP server on port 5000.

## Running Locally
The website runs automatically with the configured workflow using:
```
python -m http.server 5000 --bind 0.0.0.0
```

## Deployment
Configured as a static site deployment serving from the root directory.
